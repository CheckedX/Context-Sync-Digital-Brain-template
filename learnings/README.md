# Learnings / 学到的知识 / Knowledge & Skills

Here you store knowledge, methods, and tips you've learned.

这里存放你学到的知识、方法论、技巧。

## Folder structure / 文件夹结构

```
learnings/
├── 2026-04/           # By month / 按月份组织
│   └── example.md
└── README.md
```

## What to put here / 适合记录的内容

| EN | 中文 |
|----|------|
| New concepts and terms | 新学的概念和术语 |
| Methods and frameworks | 方法论和框架 |
| Tool usage tips | 工具使用技巧 |
| Industry insights | 行业洞察 |
| Experience from practice | 从实践中总结的经验 |

## What NOT to put here / 不适合记录的内容

- Specific projects → `projects/` / 具体项目 → `projects/`
- Personal background → `profile/` / 个人背景 → `profile/`
- Important decisions → `decisions/` / 重要决策 → `decisions/`

## Template / 模板

```markdown
# 主题名称

## 是什么

## 为什么重要

## 怎么做

## 相关资源
```
