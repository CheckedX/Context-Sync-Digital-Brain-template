# My Context Template

这是我的 AI 记忆库模板。Fork 后填入你自己的内容即可使用。

## 快速开始

1. **Fork 这个仓库**
   - 点击右上角 `Fork`
   - 你的账号下会有一份副本

2. **修改个人信息**
   - 编辑 `profile/identity.md`
   - 把示例内容换成你自己的

3. **获取 GitHub Token**
   - 按照 [教程](../docs/03-get-token.md) 获取 Token
   - 把 Token 给你的 AI 助手

4. **开始对话**
   - 正常跟 AI 聊天
   - 说"commit"即可上传

## 文件夹说明

| 文件夹 | 用途 |
|--------|------|
| profile/ | 个人档案 |
| projects/ | 项目经验 |
| learnings/ | 学到的知识 |
| decisions/ | 重要决策 |
| idea-seeds/ | 想法种子 |
| glossary.md | 术语表 |

## 模板文件

每个文件夹里都有模板文件，直接填写即可。

## 安全提醒

- ⚠️ 这是私有仓库，只有你能看到
- ⚠️ Token 不要泄露给别人
- ⚠️ 敏感内容不要放 GitHub
