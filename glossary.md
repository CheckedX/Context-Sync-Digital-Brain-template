# 术语表

记录你学到的行业术语、专业词汇。

## 格式建议

```markdown
### 术语名称
- **是什么**：简单解释
- **例子**：具体场景
- **相关**：相关的其他术语
```

---

## 示例

### Token
- **是什么**：一把钥匙，让程序能操作你的 GitHub
- **例子**：`ghp_xxxxx...`
- **相关**：API、GitHub

### Context
- **是什么**：AI 知道的关于你的所有信息
- **例子**：你的背景、目标、偏好
- **相关**：记忆、知识库

### Commit
- **是什么**：保存一次修改到版本控制
- **例子**：把新学的内容存到 GitHub
- **相关**：Git、GitHub
