# Decisions / 重要决策 / Important Decisions

Here you store important decisions you've made, and the reasons behind them.

这里存放你做过的重要决定，以及做决定的理由。

## Why record decisions? / 为什么记录决策？

| EN | 中文 |
|----|------|
| Future reference | 未来遇到类似情况，可以回顾当时的思考 |
| Self-understanding | 看清自己的决策模式 |
| Help AI understand you | AI 知道你的决策逻辑，能更好地帮你 |

## What to put here / 适合记录的内容

| EN | 中文 |
|----|------|
| Career direction decisions | 职业方向的决策 |
| Technology choices | 技术选型的决策 |
| Lifestyle changes | 生活方式的改变 |
| Any decision worth remembering | 任何你觉得"值得记住为什么"的决定 |

## Template / 模板

```markdown
# 决策名称

## 决策内容
（一句话说明做了什么决定）

## 决策理由
1. 理由一
2. 理由二

## 考虑过的其他选项
- 选项 A：为什么没选
- 选项 B：为什么没选

## 后续验证
（过一段时间回来更新：这个决定对吗？）
```
