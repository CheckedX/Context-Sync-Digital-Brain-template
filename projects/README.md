# Projects / 项目经验 / Project Experiences

Here you store records of projects you've done or are doing.

这里存放你做过或正在做的项目记录。

## What to put here / 适合记录的内容

| EN | 中文 |
|----|------|
| Project goals and background | 项目目标和背景 |
| Your role and contribution | 你的角色和贡献 |
| Tech stack and tools | 技术栈和工具 |
| Lessons learned | 经验教训 |
| Future improvements | 后续优化方向 |

## Files / 文件说明

- `project-template.md` - Template for project records / 项目记录模板

## Template / 模板

```markdown
# 项目名称

## 项目目标

## 我的角色

## 技术栈/工具

## 关键收获

## 后续计划
```

## Not for / 不适合记录的内容

- General knowledge → `learnings/` / 通用知识 → `learnings/`
- Project ideas → `idea-seeds/` / 项目想法 → `idea-seeds/`
