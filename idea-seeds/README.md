# Idea Seeds / 想法种子 / Ideas & Inspirations

Here you store inspirations, ideas, and projects you want to do but haven't started.

这里存放你的灵感、想法、想做但还没做的项目。

This is a place for seeds — some will grow into real projects, some may be shelved. That's fine, just write them down first.

这里是种子的地方——有些会发芽长大，变成真正的项目；有些可能暂时搁置。没关系，先记下来。

## What to put here / 适合记录的内容

| EN | 中文 |
|----|------|
| Sudden ideas | 突然想到的点子 |
| Projects you want to do | 想做但还没时间做的项目 |
| Things you want to share after learning | 学会了某个技能后，想分享给更多人的想法 |
| Future plans | 对未来的规划设想 |

## Seed lifecycle / 种子的生命周期

```
idea-seeds/ → projects/
   种子          成长的项目
   seeds      grown projects
```

When an idea becomes action, move the file to `projects/` folder.

当想法变成行动，就可以把文件移动到 `projects/` 文件夹，变成正式的项目记录。

## What NOT to put here / 不适合记录的内容

- Completed projects → `projects/` / 已完成的项目 → `projects/`
- Specific knowledge → `learnings/` / 具体的知识点 → `learnings/`
